Politique de Confidentialite

Derniere mise a jour : 2 fevrier 2026

Resume : DansMonSac respecte votre vie privee. Nous collectons uniquement les donnees necessaires au fonctionnement de l'application et ne les partageons jamais avec des tiers.

1. Introduction

DansMonSac est une application mobile developpee par Kappsmobile ("nous", "notre") qui aide les eleves a preparer leur sac d'ecole en fonction de leur emploi du temps.

Cette politique de confidentialite explique comment nous collectons, utilisons et protegeons vos informations lorsque vous utilisez notre application.

2. Donnees Collectees

Nous collectons les donnees suivantes :

  • Identifiant unique de l'appareil : Un identifiant anonyme genere automatiquement pour associer vos donnees a votre appareil.
  • Donnees de cours : Les noms de cours que vous creez.
  • Fournitures scolaires : Les fournitures associees a chaque cours.
  • Emploi du temps : Les horaires et jours de cours que vous configurez.
  • Preferences de notification : L'heure a laquelle vous souhaitez recevoir les rappels.

3. Utilisation des Donnees

Vos donnees sont utilisees exclusivement pour :

  • Afficher votre emploi du temps et vos fournitures
  • Envoyer des notifications de rappel pour preparer votre sac
  • Synchroniser vos donnees entre vos appareils (si vous utilisez la fonction de partage)
  • Permettre le partage d'emploi du temps avec d'autres utilisateurs via un code

4. Stockage et Securite

Vos donnees sont stockees de maniere securisee sur les serveurs de Supabase, un service d'hebergement conforme aux normes de securite modernes.

Les donnees sont egalement stockees localement sur votre appareil pour permettre une utilisation hors ligne.

  • Chiffrement des donnees en transit (HTTPS/TLS)
  • Acces aux donnees restreint a votre appareil uniquement
  • Aucun mot de passe stocke (authentification par appareil)

5. Partage des Donnees

Nous ne vendons jamais vos donnees.

Vos donnees peuvent etre partagees uniquement dans les cas suivants :

  • Partage volontaire : Lorsque vous generez un code de partage pour partager votre emploi du temps avec un autre utilisateur.
  • Obligation legale : Si la loi nous y oblige.

6. Permissions de l'Application

L'application demande les permissions suivantes :

  • Notifications : Pour vous envoyer des rappels de preparation du sac.
  • Camera (optionnel) : Pour scanner les QR codes de partage.
  • Acces Internet : Pour synchroniser vos donnees avec le serveur.

7. Conservation des Donnees

Vos donnees sont conservees tant que vous utilisez l'application. Si vous desinstallez l'application, les donnees locales sont supprimees automatiquement.

Pour supprimer vos donnees du serveur, contactez-nous a l'adresse ci-dessous.

8. Droits des Utilisateurs

Conformement au RGPD, vous avez le droit de :

  • Acceder a vos donnees personnelles
  • Rectifier vos donnees
  • Supprimer vos donnees
  • Exporter vos donnees
  • Vous opposer au traitement de vos donnees

Pour exercer ces droits, contactez-nous.

9. Enfants

DansMonSac est destine aux eleves de tous ages. Nous ne collectons pas sciemment d'informations personnelles identifiables au-dela de ce qui est necessaire au fonctionnement de l'application.

Les parents ou tuteurs peuvent nous contacter pour toute question concernant les donnees de leurs enfants.

10. Modifications

Nous pouvons mettre a jour cette politique de confidentialite periodiquement. Les modifications seront publiees sur cette page avec une nouvelle date de mise a jour.
