Questions Frequentes

Comment ajouter un nouveau cours ?

Depuis l'ecran d'accueil, appuyez sur le bouton "+" en bas a droite, puis selectionnez "Ajouter un cours". Entrez le nom du cours et validez.

Comment configurer les notifications ?

Allez dans les parametres de l'application et selectionnez l'heure a laquelle vous souhaitez recevoir le rappel pour preparer votre sac. La notification sera envoyee chaque jour a cette heure.

Comment partager mon emploi du temps ?

Depuis l'ecran principal, appuyez sur l'icone de partage. Un code unique sera genere que vous pourrez partager avec vos amis. Ils pourront importer votre emploi du temps en entrant ce code.

Mes donnees sont-elles synchronisees ?

Oui ! Vos donnees sont automatiquement sauvegardees sur nos serveurs. Si vous reinstallez l'application sur le meme appareil, vos donnees seront restaurees automatiquement.

Comment supprimer mes donnees ?

Pour supprimer toutes vos donnees, desinstallez simplement l'application. Les donnees locales seront supprimees. Pour supprimer egalement les donnees du serveur, contactez-nous par email.

L'application ne fonctionne pas hors ligne ?

DansMonSac fonctionne completement hors ligne ! Vos donnees sont stockees localement et seront synchronisees automatiquement des que vous aurez une connexion internet.
