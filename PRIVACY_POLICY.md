# Politique de Confidentialité - DansMonSac

**Dernière mise à jour : 7 janvier 2026**

## Introduction

Bienvenue sur DansMonSac. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles lors de l'utilisation de notre application mobile.

## Collecte des données

### Données stockées localement

DansMonSac stocke toutes vos données **localement sur votre appareil**. Aucune donnée n'est envoyée à nos serveurs ou à des tiers.

Les données suivantes sont stockées sur votre appareil :

- **Emploi du temps** : Liste de vos cours, matières et horaires
- **Fournitures scolaires** : Liste des fournitures associées à chaque cours
- **Préférences** : Heure de préparation du sac, couleur d'accent, date de rentrée
- **État des tâches** : Cases cochées dans la liste de fournitures

### Données que nous NE collectons PAS

- Aucune information personnelle identifiable (nom, prénom, âge, etc.)
- Aucune donnée de localisation
- Aucune photo ou média
- Aucune donnée de navigation ou d'utilisation
- Aucune donnée analytique

## Utilisation des données

Les données stockées localement sur votre appareil sont utilisées exclusivement pour :

- Afficher votre emploi du temps personnalisé
- Générer la liste des fournitures à préparer
- Envoyer des notifications de rappel
- Sauvegarder vos préférences

## Partage des données

**Nous ne partageons aucune donnée avec des tiers**, car nous ne collectons aucune donnée.

Toutes les informations restent sur votre appareil et ne quittent jamais celui-ci.

## Sécurité des données

- Les données sont protégées par les mécanismes de sécurité de votre appareil (chiffrement du système, verrouillage d'écran, etc.)
- Nous n'avons pas accès à vos données
- Aucune connexion internet n'est requise pour le fonctionnement de l'application (sauf pour les notifications)

## Notifications

L'application utilise les notifications locales pour vous rappeler de préparer votre sac. Ces notifications sont gérées directement par votre appareil et ne transmettent aucune donnée à nos serveurs.

**Permissions requises :**
- **Notifications** : Pour envoyer des rappels quotidiens
- **Alarmes exactes** : Pour programmer les notifications à l'heure précise choisie

## Suppression des données

Pour supprimer toutes vos données :

1. Désinstallez l'application de votre appareil
2. Toutes les données seront automatiquement supprimées

Alternativement, vous pouvez effacer les données de l'application depuis les paramètres de votre appareil :
- **Android** : Paramètres > Applications > DansMonSac > Stockage > Effacer les données
- **iOS** : Désinstaller l'application

## Droits des utilisateurs

Conformément au RGPD (Règlement Général sur la Protection des Données), vous disposez des droits suivants :

- **Droit d'accès** : Toutes vos données sont accessibles directement depuis l'application
- **Droit de rectification** : Vous pouvez modifier vos données à tout moment dans l'application
- **Droit à l'effacement** : Désinstallez l'application pour supprimer toutes les données
- **Droit à la portabilité** : Vos données restent sur votre appareil

Étant donné que nous ne collectons aucune donnée personnelle, ces droits s'exercent directement via l'application sans avoir besoin de nous contacter.

## Services tiers

DansMonSac n'utilise aucun service tiers de suivi, d'analyse ou de publicité.

L'application utilise uniquement :
- **Système de notifications local** : Fourni par Android/iOS pour les rappels
- **Stockage local** : SharedPreferences (Android) / UserDefaults (iOS)

## Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Toute modification sera publiée sur cette page avec une nouvelle date de "dernière mise à jour".

Les changements importants seront signalés via une notification dans l'application.

## Public visé

Cette application est destinée aux élèves de collège et lycée. Nous recommandons la supervision parentale pour les utilisateurs de moins de 13 ans.
